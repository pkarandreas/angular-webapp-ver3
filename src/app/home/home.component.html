  • Creating and using Components
  • Creating and using Modules
  • Using pipes

  • Using D.I. Dependency Injection
  • Using Observables and rxjs library
  • Using @Input() Directive

  • Accessing WEB A.P.I. using http
  • Creating and using Template Driven Forms
  • Creating and using Reactive Forms

  • Creating and Using Routing
  • Install and using Bootstrap in Angular
  • Installing and using JQuery
  • Install and User Angular/Material
  • Deploy Angular project
